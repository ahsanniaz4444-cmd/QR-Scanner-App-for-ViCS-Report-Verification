A Progressive Web Application (PWA) developed to streamline vehicle inspection and certificate verification. The system enables authorized users to securely manage vehicle inspection records, verify certificates, and access real-time data through a responsive and user-friendly interface.

Features
Secure User Authentication
Vehicle Certificate Verification
QR Code Scanning
Inspection Record Management
Responsive PWA Design
Real-Time Database with Supabase
Tech Stack
HTML5
CSS3
JavaScript
Tailwind CSS
Supabase
Progressive Web App (PWA)
Getting Started
Clone the repository.
Configure your Supabase credentials.
Run the project using a local web server.
